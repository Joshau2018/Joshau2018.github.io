Step Five in Planning for Making an AI

After completing step 4, you might be wondering how you are going to create the front end, or user interface.

If you want to use Hypertext Markup Language (HTML) with some JavaScript (JS) and Cascading Style Sheets (CSS) to create a web page for your AI to interact with, then I recommend finding a tutorial on YouTube with a design you like. After you find a design, you can base the design of your front end on it by looking at the documentation and revising the page to include the API you created.

If you want to use a different approach, like using Swift, Flutter, or React Native to create a mobile interface, then depending on the device you’re building for, you will need to consider which tool would work best.

If the device is iOS, then Swift, React Native, and Flutter all work. Swift is the primary language used for iOS development. The tools you’ll use with Swift are:

  • SwiftUI: A modern UI toolkit where developers describe what the UI should look like. It has limited backward compatibility but uses concise and easy-to-understand code (medium.com).
  • UIKit: The original UI framework where developers specify step-by-step instructions to create and update UI components. It has full backward compatibility but a more complex and difficult-to-understand code structure (medium.com).

If you want to make it cross-platform, then React Native and Flutter are great options.

  • React Native: A JavaScript library for building user interfaces (reactnative.dev). Some applications built using React Native include Discord, Microsoft Teams, Xbox App, PlayStation App, and the Amazon mobile app (reactnative.dev).
  • Flutter: Uses the Dart programming language and was developed by Google (amazon.com). Flutter is open source and supported by Google. It's used in applications like Google Classroom, Google Pay, Supercell, Universal Studios, PUBG Mobile, and eBay.

Regardless of which front end you choose, reviewing examples of other user interfaces can help you decide how to design your own. After deciding on your front end design, you’ll need to figure out how to integrate your API so information can pass between the user and the AI. This decision is based entirely on personal preference.

This concludes the planning process, and now you’re ready to start building your AI.
